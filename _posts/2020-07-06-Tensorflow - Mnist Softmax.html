overfitting¶

분류문제를 할때 outlier 마저 cover하는 곡선(곡면)이 그려진다면, overfitting이 될 가능성이 크다. 즉, 일반적으로 고양이로 분류되는 input이 overfitting 된 곡면에 의하여 개로 분류될 수 있다.

error가 y축 훈련시간이 x축일때, test data를 넣으면 훈련시간이 증가할 수록 error가 줄어들지만 특정 threshold를 넘으면 error가 다시 증가하는 경우게 생긴다. 이를 overfitting이라고 한다.

Mnist¶

Mnist는 28 x 28 크기의 이미지를 1차원으로 784개의 원소를 가진 벡터로 만들고 이것이 인풋이 된다. 즉 인풋 x_i 의 개수가 784개가 된다.

output으로는 y_i 가 10개 나온다 이는 0~9까지로 예측될 가능성(likelihood)이다.

그래프 구성하기¶

In [0]:
X = tf.placeholder(tf.float32, [None, 784])
Y = tf.placeholder(tf.float32, [None, 10])
W = tf.Variable(tf.zeros([784, 10])) # 여기서 784 == 28 * 28, 즉 input data의 행렬 크기
b = tf.Variable(tf.random_normal([10]))
Yhat = tf.matmul(X, W)+b # Yhat 도 60000 x 10 의 크기를 가짐

hypothesis=tf.nn.softmax(Yhat) # 가장 높은 확률을 가진 숫자 (인덱스) 를 뽑아줌 즉  [0.2,0.8] 을 이면 [0,1]이 됨
correct=tf.equal(tf.argmax(hypothesis,1),tf.argmax(Y,1)) #여기서 가장 높은 값에 대한 인덱스를 가져올거임. argmax의 두번째 인자는 axis=1 즉 한 행안에서의 연산이라는 소리임
                                                        # tf.equal함수는 상응하는 원소끼리 비교해서 일치하면 True, 다르면 False이고 
accuracy=tf.reduce_mean(tf.cast(correct,tf.float32))    # True False는 평균을 낼수 없으므로 tf.cast를 이용해서 tf.float32으로 0, 1으로 바꾼 후 tf.reduce_mean으로 평균값을 구함
cost = tf.reduce_mean(tf.nn.softmax_cross_entropy_with_logits_v2(logits=Yhat, labels=Y))
train=tf.train.GradientDescentOptimizer(0.5).minimize(cost) # 0.5는 learning rate임

train data로 학습하기¶

In [12]:
training_epochs = 15 # 총 train set을 몇번 볼것인지. 즉 전체 데이터를 한번 보는것을 1 epoch 이라 하고. 여기서는 전체 데이터를 15번 보겠다는 것이다.
batch_size = 100 #한번의 epoch마다 전체 데이터 에서 100개 씩 나누어 업데이트 하겠다는 것이다. 즉 1번의 epoch마다 이 예제에서는 600번을 돌겠다는 것이다.
                  # local minimum을 벗어나기 위하여 주로 사용한다. 이를 stochastic gradient descent 라고 한다.
total_batch = x_train.shape[0] // batch_size # 600이 나올것이다.
sess=tf.Session()
sess.run(tf.global_variables_initializer())
for epoch in range(training_epochs):    
    for i in range(total_batch):
        batch_x = x_train[batch_size*i:batch_size*(i+1),:]
        batch_y = y_train[batch_size*i:batch_size*(i+1),:]
        sess.run(train,{X:batch_x,Y:batch_y})
    print(sess.run(accuracy,{X:x_train,Y:y_train}))
    print(sess.run(accuracy,{X: x_test, Y: y_test}))
